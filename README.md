# Neptun-Plus
Neptun Plus, egy unofficial, open source kliens a Neptun tanulmányi rendszerhez.
A cél, hogy egy egyszerűbb kezelőfelületet hozzunk létre a Neptunhoz. Jelenleg egy webes és egy androidos kliens van aktívan fejlesztés alatt.
## Újdonságok
Ha szeretnél értesülni arról, hogy hogyan halad a projekt, akkor csatlakozz a [Discord](https://t.co/hYFuGevuND) szerverünkhöz!
